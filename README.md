# Blog CRUD API

This project is a simple CRUD API for a blogging platform built with Node.js and MongoDB.

## Technologies
- Node.js
- Express.js
- MongoDB
- Mongoose
- HTML / JavaScript

## Features
- Create, Read, Update, Delete blog posts
- Data validation and error handling
- Simple frontend interface

## API Endpoints
- POST /blogs
- GET /blogs
- GET /blogs/:id
- PUT /blogs/:id
- DELETE /blogs/:id

## Setup
1. Install dependencies:
npm install

2. Create a `.env` file:
PORT=5000
MONGO_URI=your_mongodb_connection_string

3. Run server:
npm run dev

4. Open:
http://localhost:5000
